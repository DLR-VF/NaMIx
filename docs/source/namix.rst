User reference
==============

User reference for the NaMIx package.

This guide covers usage of all public modules and functions. Every function can be accessed via `namix.module_name.function_name()` and the vast majority of them can also be accessed directly via `namix.function_name()` as a shortcut. Only a few less-common functions are accessible only via `namix.module_name.function_name()`.
